<h1 align="center">Welcome to mydb 👋</h1>

> Very easy and quick to use db, which is weekly updated!

### 🏠 [Homepage](https://github.com/falseCloud/mydb)

## Prerequisites

- npm >=5.5.0
- node >=14.0.0

## Install

```sh
npm i
```

## Run tests

```sh
node test.js
```
